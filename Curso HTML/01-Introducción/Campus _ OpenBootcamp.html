Todos los documentos HTML comienzan y tienen q estar englobados en una etiqueta que se llama html

Después hay dos elementos importantes cabeza (head) y cuerpo (body)

Cabeza: Dentro de la cabeza va todo lo relacionado con la identidad de esta página web (título, descripción, etc). Toda la info q nosotros le vamos a dar a nuestro navegador. Cuanta más info le demos mejor.

Cuerpo: es donde vamos a crear todos los elementos q queremos q se vean.
